From Curiosity to Code: My Tech Story So Far When I began my Computer Engineering degree, I was deeply curious about how websites worked behind the scenes. That curiosity turned into late nights exploring HTML, CSS, JavaScript, and eventually diving into frameworks like React and backend tools like Node.js. My passion evolved through personal projects, challenges, and hands-on learning—especially through my Full Stack Web Development Internship at IIT Jammu, where I contributed to real-world applications that helped monitor systems and visualize data effectively. Blog Post 2

When Theory Became Practice Operating System concepts can feel abstract until you apply them. That’s exactly what I did with my project on the NRU (Not Recently Used) Page Replacement Algorithm. Written in C++, it simulates how the operating system manages memory and makes decisions about which pages to replace. This project challenged me to think like a systems programmer and gave me a newfound appreciation for how computers manage resources efficiently. It’s one of my favorite ways of blending academics with real implementation.

Blog Post 3

Turning User Needs into Tech Solutions Managing hostel room allocations manually can be messy and inefficient. So I built a Hostel Accommodation System — a C++ project focused on organizing student information, automating room assignments, and ensuring clarity for both students and wardens. This taught me how important user-centered logic is and how clean data flow leads to better systems. It wasn’t just about writing code—it was about solving a practical problem, which made it deeply satisfying.

Blog Post 4

I worked with a team to build internal dashboards, learned best practices in version control, responsive design, and backend integration. More importantly, I understood the importance of clean code, teamwork, and scalable thinking.
